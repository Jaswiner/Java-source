8大基础类型及其包装类

基础类型 占用大小 包装类型
byte 8位 Byte
short 16位 Short
int 32位 Integer
long 64位 Long
float 32位 Float
double 64位 Double
boolean 2位 Boolean
char 16位 Character

区别

  • 基础类型存储在栈上，包装类型(对象)存储在堆上

自动装箱&自动拆箱

Java SE5之后支持自动装箱和拆箱，代码：

Integer a = 5;

int b = a;

用javap对class文件进行反编译后发现，

Integer a = 5;

// 实质上
Integer a = Integer.valueOf(5);


int b = a;

// 实质上
int b = a.intValue();
