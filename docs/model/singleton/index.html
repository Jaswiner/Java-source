Java中的单例模式

  • Java版本：1.8

引子

单例模式就是在整个Java程序(进程)中当前类只有一个实例，在全局使用的情况下只需要创建一个实例就好类，就不需要频繁的创建对象了。

实现

  • 构造函数私有(保证自己创建自己的实例)
  • 判断线程中是否有单例类的实例，如果有就返回，没有就创建

实例

Unsafe单例类
